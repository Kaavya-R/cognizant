bootstrap/ ├── css/ │ ├── bootstrap.css │ ├── bootstrap.min.css │ └── bootstrap.rtl.min.css ├── js/ │ ├── bootstrap.bundle.js │ └── bootstrap.bundle.min.js └── icons/ (optional if downloaded separately) ├── bootstrap-icons.css └── fonts/ └── *.woff2 (icon font files) css/ Contains the compiled CSS files. bootstrap.css: Unminified version (easier to read and debug). bootstrap.min.css: Minified version (smaller size for production). bootstrap.rtl.min.css: Right-to-left (RTL) language support. js/ Contains Bootstrap JavaScript plugins. bootstrap.bundle.js includes Bootstrap’s JavaScript + Popper.js (used for tooltips, dropdowns, etc.).
